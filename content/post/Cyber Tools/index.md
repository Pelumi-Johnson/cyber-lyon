## Overview

This page is a curated collection of cybersecurity resources organized by category. It serves as a reference space for tools news learning platforms and opportunities that support continuous growth in cybersecurity knowledge.

## Cyber Tools

- **Active Countermeasures Free Tools**  
  A collection of free and open source defensive security tools.

- **Ciphey**  
  Automates decryption and decoding of encodings ciphers hashes and cryptographic formats.

- **CVSS v3 Calculator**  
  Official vulnerability scoring calculator maintained by FIRST.

- **Crontab.guru**  
  Simple editor and explainer for cron scheduling expressions.

- **CVE.org**  
  The official catalog of publicly disclosed vulnerabilities.

- **CyberChef**  
  Web based tool for encoding decoding analyzing and transforming data.

- **MITRE D3FEND Matrix**  
  A defensive framework mapping countermeasures to security techniques.

- **dnstwister**  
  Domain monitoring tool for detecting typo squatting and phishing domains.

- **EmailRep**  
  Evaluates email reputation using breach data metadata and signals.

- **explainshell**  
  Explains Linux commands using manual page references.

- **Exploit Database**  
  Public archive of exploits and vulnerable software.

- **FreeFormatter**  
  Online formatting validation encoding and decoding tools.

- **NIST Glossary**  
  Official glossary of cybersecurity terminology.

- **Have I Been Pwned**  
  Checks if email addresses appear in known data breaches.

- **IPAddressGuide**  
  CIDR and IP address calculation tools.

- **JSON Lint**  
  Validates and formats JSON data.

- **MITRE ATT&CK**  
  Knowledge base of adversary tactics and techniques.

- **MyEventLog**  
  Searchable explanations for Windows event logs and syslog entries.

- **Pagodo**  
  Automates Google dorking for identifying exposed web resources.

- **PimEyes**  
  Reverse image and face recognition search engine.

- **Talos Intelligence Search**  
  Threat intelligence provided by Cisco Talos.

- **Tools4noobs**  
  Online hash calculator.

- **URL Encode Decode**  
  Encodes and decodes URL formatted strings.

- **VirusTotal**  
  Analyzes files URLs domains and IPs for malicious activity.

- **Whois Lookup**  
  Domain and IP registration lookup tool.

- **Windows Security Log Encyclopedia**  
  Reference for understanding Windows security event logs.
